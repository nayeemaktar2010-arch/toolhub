# Tools Page - Calculator

This project is a simple and useful calculator created using **HTML, CSS, and JavaScript**.  
It includes basic arithmetic operations such as:

- Addition
- Subtraction
- Multiplication
- Division

The design is mobile-friendly and easy to use.

---

## Features

✔ User-friendly interface  
✔ Works on mobile and desktop  
✔ Instant calculation  
✔ Responsive layout

---

## Technologies Used

- **HTML** – structure of the page  
- **CSS** – design and layout  
- **JavaScript** – logic and calculations  

---

## How to Use

1. Type numbers in the input area.
2. Select the operation ( + , - , × , ÷ ).
3. Press the equals (=) button to get the result.
